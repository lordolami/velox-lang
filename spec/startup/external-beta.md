# External Beta Plan

## Objective

Run a tight feedback loop with 3-10 external builders before broad startup rollout.

## Beta Cohorts

1. Solo frontend developer
2. Startup team (2-5 engineers)
3. AI-agent-first builder

## Success Metrics

1. Time to first running Velox app under 30 minutes.
2. At least 80% of beta tasks completed without maintainer intervention.
3. P1 bug count trend down week-over-week.

## Weekly Loop

1. Monday: collect blockers.
2. Wednesday: ship fixes/docs updates.
3. Friday: benchmark + reliability report.
